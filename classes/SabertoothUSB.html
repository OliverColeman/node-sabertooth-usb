Class SabertoothUSB

Controls USB-enabled Sabertooth motor drivers running in Packet Serial mode.

See https://www.dimensionengineering.com/datasheets/USBSabertoothPacketSerialReference.pdf

Note: Only Checksum protection is implemented, not CRC.

constructor

  • new SabertoothUSB(path: string, baudRate?: number, address?: number): SabertoothUSB
    • Defined in index.ts:74

    Create an object to control a motor driver.

    A connection will to the motor driver will be attempted upon creation but this is asynchronous and is not available immediately after creation. If the connection failes reconnection will be attempted automatically.

    Parameters

    • path: string

      the path to the (USB) serial port. eg /dev/ttyACM0 or COM1.

    • baudRate: number = 38400

      Serial baud rate, options are 2400, 9600, 19200, 38400 and 115200

    • address: number = 128

      The address of the Sabertooth. By default, this is 128.

    Returns SabertoothUSB
